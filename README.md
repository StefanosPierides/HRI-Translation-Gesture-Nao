# HRI-Translation-Gesture-Nao
This is the code documentation for the NAO bot for HRI. It contains the essential code for server hosting the translation and gesture recognition.

The segements in main file "PythonProject" demonstrate our main working components for demonstrative purposes and the outlying code. It features descriptions to describe each portion and how it works.
Utilising Emmanual's server host to transfer to the client on choreographe which in turn talks to the NAO bot. 

The segments in the other main file "Choreographe" contain the main choreographe components which call the client and contain a file called "Hri Movements... " which contain the gestures.
